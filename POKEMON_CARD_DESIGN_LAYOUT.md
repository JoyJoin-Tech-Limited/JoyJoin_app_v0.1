## Design Philosophy

### Pokemon Card Principles
1. **Thick borders = Premium feel**
2. **Holographic effects = Special/Rare**
3. **Large artwork = Collectible value**
4. **Stats display = Game element**
5. **Corner shine = Quality indicator**

### JoyJoin Personality Cards
1. **Archetype graphic = Hero element**
2. **Tagline = Personality hook**
3. **Radar chart = Visual stats**
4. **Color variants = Personalization**
5. **Expression options = Replayability**
