## 4. 匹配质量问题

### 4.1 核心指标

- **平均匹配相似度**: 97.79% (过高，说明原型太相似)
- **平均区分边际**: 0.484% (过低，几乎随机)
- **低区分度匹配**: 87.9% (严重问题)

### 4.2 问题解释

当用户的匹配边际<1%时，意味着第一名和第二名原型的相似度差距极小，答案的微小变化就可能改变结果，导致测试重复性差。

// 实验组: 欧氏距离 (取负值后比较)
function euclideanDistance(a, b) {
  let sum = 0;
  for (let i = 0; i < a.length; i++) {
    sum += Math.pow(a[i] - b[i], 2);
  }
  return -Math.sqrt(sum); // 负值，越大越相似
}
