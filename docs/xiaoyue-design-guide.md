# 小悦（Xiaoyue）形象设计指南
## Design Guide for Character Artist

---

## 1. 项目背景

### 1.1 平台介绍
**悦聚 JoyJoin** 是一个社交平台，主打4-6人小局社交活动。平台有一套"12原型动物社交气质系统"，用不同动物代表不同社交性格。

### 1.2 小悦的角色
小悦是平台的 **AI社交助手**，用户通过与小悦对话完成：
- 注册和资料填写
- 性格测试
- 活动匹配推荐

### 1.3 设计目标
创造一个让用户感到 **可信赖、松弛、有趣** 的AI助手形象，同时与平台的动物原型系统保持统一。

---

## 2. 人设定位

### 2.1 核心人设
**"街头老狐狸"**

> 混迹社交场合多年，什么人都见过，什么场面都能接住。
> 表面玩世不恭，实际上比谁都靠谱。

### 2.2 性格关键词
| 关键词 | 说明 |
|--------|------|
| **松弛** | 不刻意、不用力，自然放松 |
| **可靠** | 表面玩世不恭，内里靠谱 |
| **聪明** | 见多识广，懂人懂事 |
| **有故事感** | 像个有阅历的老朋友 |

### 2.3 不是什么
- ❌ 不是热情过度的客服
- ❌ 不是冷冰冰的机器人
- ❌ 不是可爱萌系吉祥物
- ❌ 不是严肃的专家学者

---

## 3. 视觉规格

### 3.1 风格定义
| 项目 | 规格 |
|------|------|
| **风格** | 3D日式动漫风格渲染 |
| **参考** | 《疯狂动物城》Nick Wilde 的质感 |
| **形象** | 拟人化狐狸（不是人类+狐狸耳朵） |
| **年龄感** | 年轻成熟（25-30岁人类等效） |

### 3.2 基础外观

#### 毛发
- **颜色**：暖橙棕色（#D4845C 参考）
- **质感**：自然柔顺，略有光泽
- **注意**：不要蓬松乱毛，保持整洁

#### 耳朵
- 标准狐狸立耳
- 自然竖立，略微外倾
- **不要**：耳边白毛等"显老"细节

#### 尾巴
- **状态**：放松卷曲（95%用户选择）
- 不是下垂，不是摇动
- 自然地在身体一侧卷曲

---

## 4. 表情与眼神（最重要）

### 4.1 表情
| 项目 | 规格 | 用户支持率 |
|------|------|-----------|
| **整体** | 慵懒放松 | 57% |
| **嘴角** | 淡淡的、不用力的微笑 | - |
| **感觉** | "我知道些什么"但不刻意 | - |

### 4.2 眼神（关键设计点）
| 项目 | 规格 | 用户支持率 |
|------|------|-----------|
| **神态** | 沉稳柔和 | 89% |
| **方向** | 微微向下看（不直视） | 7.9/10 |
| **AI标识** | 瞳孔深处有微弱紫色光环 | - |

> **用户反馈**：
> "沉稳柔和的眼神是'表面玩世不恭实际靠谱'人设最关键的视觉锚点"

### 4.3 眼神设计细节
```
瞳孔设计：
- 基础：自然的狐狸眼睛，琥珀色
- AI标识：瞳孔深处有一圈非常淡的紫色/青色光环
- 强度：若隐若现，不是明显发光
- 效果：暗示AI身份，但不破坏自然感
```

### 4.4 避免的眼神
- ❌ 机灵有神（太精明，有压迫感）
- ❌ 直视镜头（对社恐用户有压力）
- ❌ 眼角纹路（79人认为反效果，显老）

---

## 5. 姿态与动作

### 5.1 身体姿态
| 项目 | 规格 | 效果评分 |
|------|------|----------|
| **站姿** | 放松自信，略微侧身 | - |
| **头部** | 微微侧倾（倾听姿态） | **8.9/10** 🥇 |
| **重心** | 略微偏向一侧，不是正站 | - |

### 5.2 手部动作
| 项目 | 规格 | 效果评分 |
|------|------|----------|
| **左手** | 插在卫衣口袋里 | - |
| **右手** | 露出做放松的"放心"手势 | **7.9/10** 🥉 |

> **手势参考**：轻松的OK手势，或微微张开表示"交给我"

### 5.3 姿态关键点
- ✅ 头微倾传达"我在认真听你说"
- ✅ 眼神向下减少压迫感
- ✅ 手势传达"放心交给我"

---

## 6. 服装设计

### 6.1 主体服装
| 项目 | 规格 | 用户支持率 |
|------|------|-----------|
| **款式** | 连帽卫衣（hoodie） | - |
| **颜色** | 紫色（品牌色） | - |
| **质感** | 轻微做旧 | **100%** |

### 6.2 做旧细节
```
做旧程度：轻微（不是破烂）
- 袖口/下摆有自然的使用痕迹
- 颜色略有不均匀的自然感
- 像穿了一段时间但保养得不错
```

### 6.3 Logo
- **位置**：左胸口
- **大小**：适中，不要太大
- **设计**：悦聚logo（简约版）

### 6.4 避免的服装处理
- ❌ 全新崭新的感觉
- ❌ 明显的破洞/补丁
- ❌ 科技感流光线条（测试中被否决）

## 7. 配饰设计

### 7.1 配饰清单
| 配饰 | 说明 | 用户选择率 |
|------|------|-----------|
| **墨镜** | 挂在领口（不戴脸上） | **84%** ✅ |
| **手表** | 复古皮表 | 100人选择 |
| **项链** | 简约银色项链 | 84人选择 |

### 7.2 墨镜设计
```
款式：经典款，不要太夸张
位置：挂在卫衣领口
状态：一边挂着，自然下垂
效果：增加型格，不遮挡眼神设计
```

> **用户反馈**：
> "墨镜挂领口既保留街头慵懒感，又不会像普通眼镜那样强化距离感"

### 7.3 手表设计
```
款式：复古皮表（不是智能表）
表盘：简约圆形
表带：棕色皮革
效果：传达"见过世面"的阅历感
```

### 7.4 项链设计
```
款式：极简细链
吊坠：无，或极小的几何形状
材质：银色金属
效果：低调品味，不抢眼
```

## 8. 背景与氛围

### 8.1 背景建议
| 项目 | 说明 |
|------|------|
| **风格** | 虚化/模糊处理 |
| **元素** | 隐约的咖啡店/书架轮廓 |
| **色调** | 暖色调，与角色协调 |
| **效果** | 暗示"常混社交场合" |

### 8.2 光效
```
整体光效：柔和暖光
光源方向：略微侧上方
阴影：柔和自然，不要硬边阴影
氛围：温暖、舒适、不刺眼
```

---

## 9. 整体氛围

### 9.1 氛围关键词（按用户投票排序）
1. **松弛** (35人)
2. **可靠** (23人)
3. **有故事感** (12人)
4. **不刻意** (10人)
5. **无压力** (7人)

### 9.2 整体感觉
> 一个在社交场合游刃有余的老手，
> 看起来很松弛但你知道他什么都懂，
> 不会给你压力但会在关键时刻靠谱地帮你。

---

## 10. 设计数据支撑

本设计指南基于 **5轮模拟用户测试**，累计 **500人次** 反馈：

| 测试项目 | 结果 |
|----------|------|
| 拟人狐狸 vs 人类+狐耳 | 狐狸胜 86% |
| 自然版 vs AI科技版 | 自然版胜 66% |
| 8个信任感细节测试 | 确定Top 3 |
| 眼镜款式测试 | 墨镜挂领口 84% |
| 服装风格测试 | 轻微做旧 100% |

## 11. 交付物清单

### 11.1 必需交付
- [ ] 正面3/4视角立绘（主视觉）
- [ ] 纯正面视角立绘
- [ ] 头像版本（圆形裁切适用）
- [ ] 表情变体：微笑、思考、惊讶

### 11.2 可选交付
- [ ] 多角度视图（设定稿）
- [ ] 动作变体
- [ ] 不同表情的头像包

### 11.3 文件格式
- PNG（透明背景）
- 分辨率：至少 2000x3000px
- 色彩模式：sRGB
